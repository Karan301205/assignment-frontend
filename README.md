# Assignment Management System - Frontend

A modern, responsive React-based dashboard built for teachers and students to manage academic tasks efficiently.

## 🚀 Tech Stack
* **React.js**: Functional components and Hooks (`useState`, `useEffect`, `useContext`).
* **Tailwind CSS**: Utility-first styling for a clean, mobile-responsive UI.
* **Lucide React**: Clean and consistent iconography.
* **Axios**: Promised-based HTTP client for API communication.

## 📱 Features
* **Teacher Dashboard**: Create drafts, publish assignments, and view student submissions in real-time.
* **Student Portal**: View active assignments and submit work through an intuitive interface.
* **Fully Responsive**: Optimized for mobile, tablet, and desktop viewing.
* **Protected Routes**: Ensures only logged-in users can access the dashboards.

## 🚦 Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Update `src/services/api.js` with your backend URL.
4. Run `npm start` to launch the development server.
